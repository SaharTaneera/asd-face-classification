# Dataset Notice

This repository does not include the dataset used in the experiments.

The experiments were originally conducted using the **Autistic Children Facial Image Dataset**
published on Kaggle by **Gerry (2020)**.

The dataset is not redistributed here due to privacy considerations and because
it is no longer publicly available at the time of this repository release.

This repository is shared for academic transparency and documentation of the
experimental workflow related to the associated publication.
